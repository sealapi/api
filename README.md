#the seal api

A simple, beautiful webpage that displays a random seal image every time you visit — inspired by the Cat API.

## demo

Live at: `https://seal.github.io`

## features

- Random seal image on every load
- Click **new seal ↺** to fetch another one
- Smooth entrance animations on startup
- Calm, minimal design with a dark background
- Species name shown below each image

## seals included

- Harbour seal
- Grey seal
- Elephant seal
- Leopard seal
- Harp seal pup
- Weddell seal
- Fur seal
- Ringed seal

## usage

Just open `index.html` in a browser — no dependencies, no build step, no server required.

## hosting on github pages

1. Create a new **public** repository on [github.com](https://github.com)
2. Upload `index.html` to the root of the repo
3. Go to **Settings → Pages**
4. Set the branch to `main` and folder to `/ (root)`
5. Click **Save** — your site will be live in ~30 seconds

## tech stack

- Plain HTML, CSS, JavaScript
- [Nunito](https://fonts.google.com/specimen/Nunito) via Google Fonts
- No frameworks, no dependencies
